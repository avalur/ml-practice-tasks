# Null Vector

**Topic:** `numpy_basics` &nbsp;|&nbsp; **Difficulty:** easy

Implement `null_vector(n)`.

Return a 1D array of `n` zeros (dtype float). No `for`/`while` loops.

## Constraints

- Forbidden modules: scipy, sklearn
- Explicit `for`/`while` loops are not allowed (vectorize it)

## How to run

```bash
pytest tasks/numpy_basics/null_vector
```
Edit `submission.py` until every test passes.
