# Reverse a Vector

**Topic:** `numpy_basics` &nbsp;|&nbsp; **Difficulty:** easy

Implement `reverse_vector(x)`.

Given a 1D array `x`, return its elements in reverse order (last element
first). Use slicing — no `for`/`while` loops.

## Constraints

- Explicit `for`/`while` loops are not allowed (vectorize it)

## How to run

```bash
pytest tasks/numpy_basics/reverse_vector
```
Edit `submission.py` until every test passes.
