# Transpose a Matrix

**Topic:** `numpy_basics` &nbsp;|&nbsp; **Difficulty:** easy

Implement `transpose_2d(X)`.

Given a 2D array `X` of shape `(m, n)`, return its transpose of shape
`(n, m)`: the element at position `(j, i)` of the result equals `X[i, j]`.
No `for`/`while` loops.

## Constraints

- Explicit `for`/`while` loops are not allowed (vectorize it)

## How to run

```bash
pytest tasks/numpy_basics/transpose_2d
```
Edit `submission.py` until every test passes.
