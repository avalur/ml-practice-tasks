# Mean of a Vector

**Topic:** `numpy_basics` &nbsp;|&nbsp; **Difficulty:** easy

Implement `vector_mean(x)`.

Return the arithmetic mean (average) of the 1D array `x`. (numpy-100 motivates
this by creating a random vector of size 30 and finding its mean — here you
just compute the mean of the given vector.) No `for`/`while` loops.

## Constraints

- Explicit `for`/`while` loops are not allowed (vectorize it)

## How to run

```bash
pytest tasks/numpy_basics/vector_mean
```
Edit `submission.py` until every test passes.
