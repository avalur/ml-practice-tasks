# Repeat Each Element

**Topic:** `numpy_combine` &nbsp;|&nbsp; **Difficulty:** easy

Implement `repeat_each(x, k)`.

Given a 1D array `x` and an integer `k`, return a 1D array in which each
element of `x` is repeated `k` times consecutively. For example
`repeat_each([1, 2, 3], 2)` → `[1, 1, 2, 2, 3, 3]` (not `[1, 2, 3, 1, 2, 3]`).
No `for`/`while` loops.

## Constraints

- Explicit `for`/`while` loops are not allowed (vectorize it)

## How to run

```bash
pytest tasks/numpy_combine/repeat_each
```
Edit `submission.py` until every test passes.
