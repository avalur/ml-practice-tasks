# Common Values

**Topic:** `numpy_indexing` &nbsp;|&nbsp; **Difficulty:** medium

Implement `intersect(a, b)`.

Given two 1D arrays `a` and `b`, return the sorted array of unique values
that appear in **both**. No `for`/`while` loops.

## Constraints

- Forbidden modules: scipy, sklearn
- Explicit `for`/`while` loops are not allowed (vectorize it)

## How to run

```bash
pytest tasks/numpy_indexing/intersect
```
Edit `submission.py` until every test passes.
