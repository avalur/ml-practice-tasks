# Nearest Value in a Matrix

**Topic:** `numpy_indexing` &nbsp;|&nbsp; **Difficulty:** easy

Implement `nearest_value(matrix, value)`.

Return the element of `matrix` (any shape) closest to `value` — the one
minimizing `|element - value|` — as a float. If the matrix is empty, return
`None`. No `for`/`while` loops.

## Constraints

- Explicit `for`/`while` loops are not allowed (vectorize it)

## How to run

```bash
pytest tasks/numpy_indexing/nearest_value
```
Edit `submission.py` until every test passes.
