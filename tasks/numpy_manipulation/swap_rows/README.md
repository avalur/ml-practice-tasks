# Swap Two Rows

**Topic:** `numpy_manipulation` &nbsp;|&nbsp; **Difficulty:** easy

Implement `swap_rows(X, i, j)`.

Given a 2D array `X` and two row indices `i`, `j`, return a **new** array
equal to `X` with rows `i` and `j` swapped (leave `X` unchanged). No
`for`/`while` loops.

## Constraints

- Forbidden modules: scipy, sklearn
- Explicit `for`/`while` loops are not allowed (vectorize it)

## How to run

```bash
pytest tasks/numpy_manipulation/swap_rows
```
Edit `submission.py` until every test passes.
