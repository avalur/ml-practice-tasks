# Unique Rows

**Topic:** `numpy_warmup` &nbsp;|&nbsp; **Difficulty:** medium

Implement `unique_rows(X)`.

Given a 2D array `X`, return its distinct rows as a 2D array, sorted in
ascending (lexicographic) order — i.e. the result of treating each row as a
unit. No `for`/`while` loops.

## Constraints

- Forbidden modules: scipy, sklearn
- Explicit `for`/`while` loops are not allowed (vectorize it)

## How to run

```bash
pytest tasks/numpy_warmup/unique_rows
```
Edit `submission.py` until every test passes.
