# Flatten an Array

**Topic:** `numpy_basics` &nbsp;|&nbsp; **Difficulty:** easy

Implement `flatten(X)`.

Given an array `X` of any shape, return a 1D array containing all of its
elements in row-major (C) order — the first row, then the second, and so on.
No `for`/`while` loops.

## Constraints

- Explicit `for`/`while` loops are not allowed (vectorize it)

## How to run

```bash
pytest tasks/numpy_basics/flatten
```
Edit `submission.py` until every test passes.
