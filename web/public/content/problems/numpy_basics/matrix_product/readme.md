# Matrix Product

**Topic:** `numpy_basics` &nbsp;|&nbsp; **Difficulty:** easy

Implement `matrix_product(a, b)`.

Given a matrix `a` of shape `(m, k)` and a matrix `b` of shape `(k, n)`,
return their matrix product `a @ b` (shape `(m, n)`). For example, a 5Ă—3
matrix times a 3Ă—2 matrix gives a 5Ă—2 matrix. No `for`/`while` loops.

## Constraints

- Explicit `for`/`while` loops are not allowed (vectorize it)

## How to run

```bash
pytest tasks/numpy_basics/matrix_product
```
Edit `submission.py` until every test passes.
