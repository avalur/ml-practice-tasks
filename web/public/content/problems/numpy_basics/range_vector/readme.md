# Range Vector

**Topic:** `numpy_basics` &nbsp;|&nbsp; **Difficulty:** easy

Implement `range_vector(lo, hi)`.

Return a 1D array of the consecutive integers from `lo` up to but not
including `hi` (`lo, lo+1, ..., hi-1`). For example, `range_vector(10, 50)`
ranges from 10 to 49. No `for`/`while` loops.

## Constraints

- Forbidden modules: scipy, sklearn
- Explicit `for`/`while` loops are not allowed (vectorize it)

## How to run

```bash
pytest tasks/numpy_basics/range_vector
```
Edit `submission.py` until every test passes.
