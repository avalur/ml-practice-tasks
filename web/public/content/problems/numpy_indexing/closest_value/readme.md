# Closest Value

**Topic:** `numpy_indexing` &nbsp;|&nbsp; **Difficulty:** easy

Implement `closest_value(x, v)`.

Given a 1D array `x` and a scalar `v`, return the element of `x` closest to
`v` (the one minimizing `|x - v|`). No `for`/`while` loops.

## Constraints

- Forbidden modules: scipy, sklearn
- Explicit `for`/`while` loops are not allowed (vectorize it)

## How to run

```bash
pytest tasks/numpy_indexing/closest_value
```
Edit `submission.py` until every test passes.
