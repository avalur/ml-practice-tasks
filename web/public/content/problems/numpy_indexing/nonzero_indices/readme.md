# Indices of Non-Zero Elements

**Topic:** `numpy_indexing` &nbsp;|&nbsp; **Difficulty:** easy

Implement `nonzero_indices(x)`.

Given a 1D array `x`, return a 1D array of the indices where `x` is non-zero,
in increasing order. No `for`/`while` loops.

## Constraints

- Forbidden modules: scipy, sklearn
- Explicit `for`/`while` loops are not allowed (vectorize it)

## How to run

```bash
pytest tasks/numpy_indexing/nonzero_indices
```
Edit `submission.py` until every test passes.
