# Trace by Hand

**Topic:** `numpy_linalg` &nbsp;|&nbsp; **Difficulty:** easy

Implement `trace(X)`.

Return the trace of a square matrix `X` (the sum of its diagonal entries).
Compute it yourself — `np.trace` is **not allowed** — and no `for`/`while`
loops.

## Constraints

- Forbidden modules: scipy, sklearn
- Forbidden functions: trace
- Explicit `for`/`while` loops are not allowed (vectorize it)

## How to run

```bash
pytest tasks/numpy_linalg/trace
```
Edit `submission.py` until every test passes.
