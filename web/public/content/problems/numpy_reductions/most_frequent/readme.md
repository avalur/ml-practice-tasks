# Most Frequent Value

**Topic:** `numpy_reductions` &nbsp;|&nbsp; **Difficulty:** easy

Implement `most_frequent(x)`.

Given a 1D array `x` of non-negative integers, return the value that occurs
most often. If several values tie, return the smallest of them. No
`for`/`while` loops.

## Constraints

- Forbidden modules: scipy, sklearn
- Explicit `for`/`while` loops are not allowed (vectorize it)

## How to run

```bash
pytest tasks/numpy_reductions/most_frequent
```
Edit `submission.py` until every test passes.
